# Markdown 操作 (42-45)

> ⭐ 简单 | 分类：文档编辑 | 预估时间：2 分钟

### 场景描述

Markdown 是常用的文档格式，掌握这些技巧可以提高文档编写效率。

---

## 42. 创建 Markdown 链接

> 难度：⭐ | 预估时间：30 秒

### 初始状态

```text
https://github.com
```

### 目标状态

```markdown
[GitHub](https://github.com)
```

---

### 操作方法

**方法一：viwS[**

```vim
viwS[
```

**方法二：手动**

```vim
I[<Esc>E](
```

---

### 检验清单

- [ ] 能快速创建链接

## 43. 创建 Markdown 列表

> 难度：⭐ | 预估时间：30 秒

### 初始状态

```text
apple
banana
cherry
```

### 目标状态

```markdown
- apple
- banana
- cherry
```

---

### 操作方法

**方法一：可视行+替换**

```vim
V3j
:s/^/- /
```

**方法二：substitute**

```vim
:%s/^/- /
```

---

### 检验清单

- [ ] 能批量添加列表标记
- [ ] 知道可视行选中的方法

## 44. 调整标题级别

> 难度：⭐ | 预估时间：30 秒

### 初始状态

```markdown
# Title

## Subtitle
```

### 目标状态

```markdown
## Title

### Subtitle
```

---

### 操作方法

```vim
1Gyy2Gp
```

---

### 关键命令解析

| 命令 | 作用     |
| ---- | -------- |
| `1G` | 文件开头 |
| `yy` | 复制行   |
| `p`  | 粘贴     |

---

## Markdown 速查

| 操作   | 命令            | 说明       |
| ------ | --------------- | ---------- |
| 链接   | `viwS[`         | 选中后包裹 |

| 代码块 | `viwS```        | 选中后包裹 |
