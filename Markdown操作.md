## 42. 创建 Markdown 链接

```text
https://github.com
```

=>

```markdown
[GitHub](https://github.com)
```

## 43. 创建 Markdown 列表

```text
apple
banana
cherry
```

=>

```markdown
- apple
- banana
- cherry
```

## 44. 调整标题级别

```markdown
# Title

## Subtitle
```

=>

```markdown
## Title

### Subtitle
```

## 45. 创建代码块

```text
console.log("hello")
```

=>

````markdown
```javascript
console.log("hello");
```
````
