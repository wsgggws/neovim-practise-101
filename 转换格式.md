## 50. 蛇形命名转驼峰

```text
hello_world_example
user_name_field
```

=>

```text
helloWorldExample
userNameField
```

## 51. 驼峰转蛇形命名

```text
helloWorldExample
userNameField
```

=>

```text
hello_world_example
user_name_field
```

## 52. 转换为常量命名

```text
maxValue
minValue
```

=>

```text
MAX_VALUE
MIN_VALUE
```

## 53. CSV 转 JSON

```text
name,age,city
John,30,NYC
Jane,25,LA
```

=>

```json
[
  { "name": "John", "age": "30", "city": "NYC" },
  { "name": "Jane", "age": "25", "city": "LA" }
]
```
