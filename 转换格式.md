# 转换格式 (50-53)

> ⭐⭐ 中等 | 分类：数据转换 | 预估时间：3 分钟

### 场景描述

使用正则表达式进行各种命名格式的转换。

---

## 50. 蛇形命名转驼峰

> 难度：⭐⭐ | 预估时间：1 分钟

### 初始状态

```text
hello_world_example
user_name_field
```

### 目标状态

```text
helloWorldExample
userNameField
```

---

### 操作方法

```vim
:%s/_\([a-z]\)/\U\1/g
```

---

### 关键命令解析

| 部分         | 含义            |
| ------------ | --------------- |
| `_\([a-z]\)` | 匹配下划线+小写 |
| `\U\1`       | 转大写          |

---

### 检验清单

- [ ] 理解 `\U` 的作用
- [ ] 能处理多个下划线

## 51. 驼峰转蛇形命名

> 难度：⭐⭐ | 预估时间：1 分钟

```text

```

### 目标状态

### 操作方法

## 52. 转换为常量命名

> 难度：⭐ | 预估时间：30 秒

### 初始状态

```text
maxValue
minValue
```

### 目标状态

```text
MAX_VALUE
MIN_VALUE
```

---

### 操作方法

```vim
:%s/[a-z]/\U&/g
```

---

### 关键命令解析

| 部分  | 含义     |
| ----- | -------- |
| `\U&` | 转大写   |
| `&`   | 原始匹配 |

---

### 检验清单

- [ ] 知道 `\U` 的用法
- [ ] 能转换整个单词

## 53. CSV 转 JSON

> 难度：⭐⭐ | 预估时间：1 分钟

### 初始状态

```csv
name,age,city
John,30,NYC
Jane,25,LA
```

### 目标状态

```json
[
  { "name": "John", "age": "30", "city": "NYC" },
  { "name": "Jane", "age": "25", "city": "LA" }
]
```

---

### 操作方法

```vim
:%!python -c "import csv, json, sys; print(json.dumps([dict(r) for r in csv.DictReader(sys.stdin)], indent=2))"
```

---

### 检验清单

- [ ] 知道如何使用外部命令
- [ ] 能处理 CSV 数据

## 格式转换速查

| 全部大写  | `:%s/.*/\U&/`           |
