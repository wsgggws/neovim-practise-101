# 注释操作 (9-10)

> ⭐ 简单 | 分类：基础编辑 | 预估时间：1 分钟

### 场景描述

批量注释和取消注释是代码调试和文档编写中的常用操作。

---

## 9. 批量注释代码

> 难度：⭐ | 预估时间：30 秒

### 初始状态

```python
def hello():
    print("Hello")
    return True
```

### 目标状态

```python
# def hello():
#     print("Hello")
#     return True
```

---

### 操作方法

**方法一：可视行选中（推荐）**

```vim
V2jgc
```

**方法二：命令行**

```vim
:2,4s/^/#/
```

**方法三：LazyVim 快捷键**

```vim
gcc  " 注释当前行
```

---

### 关键命令解析

| 命令 | 作用                       |
| ---- | -------------------------- |
| `V`  | 进入行可视模式             |
| `2j` | 向下选中 2 行              |
| `gc` | toggle comment（切换注释） |

---

### 常见错误

- ❌ `vgc` - 可视字符模式可能注释不完整
- ✅ `Vgc` - 行可视模式注释整行

---

### 进阶扩展

**不同语言的注释符：**

```vim
" Python: #
" JavaScript: // 或 /*
" CSS: /*
" HTML: <!-- -->
```

## 10. 批量取消注释

> 难度：⭐ | 预估时间：30 秒

### 初始状态

```python
# def hello():
#     print("Hello")
#     return True
```

### 目标状态

```python
def hello():
    print("Hello")
    return True
```

---

### 操作方法

**方法一：可视行选中**

```vim
V2jgc
```

**方法二：命令行**

```vim
:1,3s/^#//
```

**方法三：LazyVim**

```vim
gcc  " 再次按取消注释
```

---

### 关键命令解析

| 命令      | 作用           |
| --------- | -------------- |
| `:s/^#//` | 删除行首 `#`   |
| `gc`      | toggle comment |

---

### 常见错误

- ❌ 删除 `#` 后忘了删除空格
- ✅ `:%s/^# *//` 同时删除 `#` 和后面的空格

---

### 进阶扩展

**只取消非空行的注释：**

```vim
:v/^#/s/^#//
```

## 注释操作速查

| 操作       | 命令          | 说明           |
| ---------- | ------------- | -------------- |
| 注释当前行 | `gcc`         | toggle comment |
| 注释选中行 | `gc`          | 可视模式下使用 |
| 行首添加   | `I#<Esc>`     | 手动注释       |
| 取消注释   | `:s/^#//`     | 删除行首 #     |
| 批量注释   | `:n,m s/^/#/` | n-m 行注释     |
| 批量取消   | `:n,m s/^#//` | n-m 行取消     |
