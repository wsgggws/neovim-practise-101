# 文本对象 (57-60)

> ⭐ 简单 | 分类：基础编辑 | 预估时间：2 分钟

### 场景描述

文本对象是 Vim 的核心概念，掌握后可以高效操作各种文本结构。

---

## 57. 操作单词

> 难度：⭐ | 预估时间：30 秒

### 初始状态

```text
hello world neovim
```

### 目标状态

```text
hello neovim
```

---

### 操作方法

**方法一：diw（推荐）**

```vim
diw
```

**方法二：daw**

```vim
daw
```

---

### 解法对比

| 方法  | 删除内容 | 保留内容 |
| ----- | -------- | -------- |
| `diw` | 单词内部 | 空格     |
| `daw` | 整个单词 | 无       |

- [ ] 理解 `iw` 的含义

## 58. 操作句子

> 难度：⭐⭐ | 预估时间：30 秒

### 初始状态

```text
First sentence. Second sentence. Third sentence.
```

### 目标状态

```text
First sentence. Third sentence.
```

---

### 操作方法

```vim
dis  " delete inside sentence
```

---

### 关键命令解析

| 命令 | 作用            |
| ---- | --------------- |
| `is` | inside sentence |
| `as` | a sentence      |

---

### 检验清单

- [ ] 知道句子的边界是 `.`

## 59. 操作段落

> 难度：⭐⭐ | 预估时间：30 秒

```text

Multiple lines here.

Paragraph two.
More content.
```

### 目标状态

```text
Paragraph two.

```

---

### 操作方法

```vim

```

---

### 关键命令解析

| 命令 | 作用             |
| ---- | ---------------- |

| `ap` | a paragraph      |

## 60. 操作函数参数

> 难度：⭐⭐ | 预估时间：1 分钟

### 初始状态

```python
function(arg1, arg2, arg3)
```

### 目标状态

```python
function(arg1, arg3)
```

---

### 操作方法

```vim
f,     " 找到逗号
d2f,   " 删除到下个逗号前
" 或
2f,di, " 第二个参数开头，删除到逗号
```

---

### 关键命令解析

| 命令   | 作用               |
| ------ | ------------------ |
| `f,`   | 找到下一个逗号     |
| `d2f,` | 删除到下下个逗号前 |

---

### 检验清单

- [ ] 能用 `f` 导航
- [ ] 知道如何处理嵌套括号

## 文本对象速查

| 对象 | 含义             | 示例               |
| ---- | ---------------- | ------------------ |
| `iw` | inside word      | `ciw` - 修改单词   |
| `i"` | inside quotes    | `ci"` - 修改引号内 |
| `i(` | inside parens    | `ci(` - 修改括号内 |
| `ip` | inside paragraph | `cip` - 修改段落   |
| `is` | inside sentence  | `cis` - 修改句子   |
| `as` | a sentence       | `das` - 删除句子   |
| `ap` | a paragraph      | `dap` - 删除段落   |

### 修改/删除/复制

| 命令  | 功能               |
| ----- | ------------------ |
| `ciw` | 修改单词           |
| `diw` | 删除单词           |
| `yi"` | 复制引号内         |
| `ca"` | 修改引号（含引号） |
| `da"` | 删除引号（含引号） |
