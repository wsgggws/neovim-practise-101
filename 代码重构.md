## 32. 提取变量

```javascript
const total = price * 0.9 + price * 0.1;
```

=>

```javascript
const discount = 0.9;
const tax = 0.1;
const total = price * discount + price * tax;
```

## 33. 重命名变量

```python
def calculate(x):
    result = x * 2
    return result
```

=>

```python
def calculate(number):
    doubled = number * 2
    return doubled
```

## 34. 函数参数对齐

```python
function_call(arg1, arg2, arg3, arg4, arg5)
```

=>

```python
function_call(
    arg1,
    arg2,
    arg3,
    arg4,
    arg5
)
```
