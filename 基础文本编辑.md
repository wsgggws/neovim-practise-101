## 3. 添加递升/递减序号

**初始状态**：

```text
1
2
3
4
```

**操作提示**：

```vim
" 递增
:let i=1 | g/^/s//\=i." "/ | let i=i+1

" 递减
:let i=4 | g/^/s//\=i." "/ | let i=i-1
```

**目标状态**：

```text
1. 1
2. 2
3. 3
4. 4

# 或递减
4. 1
3. 2
2. 3
1. 4
```

## 4. 倒置文本行

**初始状态**：

```text
1111111
3333333
2222222
4444444
```

**操作提示**：

```vim
:g/^/m 0
```

`g/^/m 0` - 将每行移动到文件开头，实现倒序

**目标状态**：

```text
4444444
2222222
3333333
1111111
```

## 5. 行排序

**初始状态**：

```text
case 1
4
2
3
1

case 2
4.1
2.4
3.2
1.3
```

**操作提示**：

```vim
" 升序
:sort n

" 降序
:sort! n
```

**目标状态**：

```text
case 1
1
2
3
4

case 2
1.3
2.4
3.2
4.1
```

---

## 6. 更改文本对象内容

**初始状态**：

```text
{"key": "value"}
```

**操作提示**：

```vim
ci"wsgggws<Esc>
```

`ci"` - Change inside quotes，先删除引号内内容，再插入新内容

**目标状态**：

```text
{"key": "wsgggws"}
```
