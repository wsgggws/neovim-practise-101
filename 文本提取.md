# 文本提取 (90-92)

> ⭐⭐ 中等 | 数据提取

## 90. 提取所有邮箱地址

**初始状态**：

```text
Contact: john@example.com
Support: support@company.com
Sales: sales@business.org
```

**操作提示**：

```vim
:g/\w@\w/yank A
" 或
:%!grep -o '[a-zA-Z0-9._%+-]*@[a-zA-Z0-9.-]*\.[a-zA-Z]*'
```

**目标状态**：

```text
john@example.com
support@company.com
sales@business.org
```

---

## 91. 提取所有 IP 地址

**初始状态**：

```text
Server 1: 192.168.1.1
Server 2: 10.0.0.5
Server 3: 172.16.0.10
```

**操作提示**：

```vim
:%!grep -oE '[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}'
```

**目标状态**：

```text
192.168.1.1
10.0.0.5
172.16.0.10
```

---

## 92. 提取所有数字

**初始状态**：

```text
Price: $99.99
Quantity: 5 items
Total: $499.95
```

**操作提示**：

```vim
:%!grep -oE '[0-9]+\.?[0-9]*'
```

**目标状态**：

```text
99.99
5
499.95
```

---

## 提取命令速查

| 操作 | 命令                                                           |
| ---- | -------------------------------------------------------------- |
| 邮箱 | `:%!grep -o '[a-zA-Z0-9._%+-]*@[a-zA-Z0-9.-]*'`                |
| IP   | `:%!grep -oE '[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}'` |
| 数字 | `:%!grep -oE '[0-9]+\.?[0-9]*'`                                |
