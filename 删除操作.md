# 删除操作 (24-28)

> ⭐ 简单 | 分类：基础编辑 | 预估时间：2 分钟

### 场景描述

删除是编辑中最基础的操作，掌握各种删除技巧能大幅提高效率。

---

## 24. 删除单词

> 难度：⭐ | 预估时间：30 秒

### 初始状态

```text
hello beautiful world
```

### 目标状态

```text
hello world
```

---

### 操作方法

**方法一：diw（推荐）**

```vim
diw
```

**方法二：daw**

```vim
daw
```

---

### 解法对比

| 解法  | 删除内容 | 保留内容 |
| ----- | -------- | -------- |
| `diw` | 单词内部 | 空格     |
| `daw` | 整个单词 | 无       |

---

### 关键命令解析

| 命令 | 作用       |
| ---- | ---------- |
| `d`  | delete     |
| `iw` | inner word |
| `aw` | a word     |

---

### 常见错误

- ❌ `dw` - 删除到下个单词开头
- ✅ `diw` - 删除整个单词

---

### 检验清单

- [ ] 理解 `i` 和 `a` 的区别
- [ ] 能在 2 秒内完成

## 27. 删除括号内内容

> 难度：⭐⭐ | 预估时间：30 秒

### 初始状态

```python
function(arg1, arg2, arg3)
```

### 目标状态

```python
function()
```

---

### 操作方法

```vim
di(
```

---

### 关键命令解析

| 命令  | 作用         |
| ----- | ------------ |
| `di(` | 删除括号内   |
| `di[` | 删除方括号内 |
| `di{` | 删除花括号内 |

---

### 常见错误

- ❌ 光标不在括号内
- ✅ 光标需要在括号内部或旁边

---

### 检验清单

- [ ] 知道各种括号对应的文本对象
- [ ] 能快速删除括号内容

## 28. 删除引号内内容

> 难度：⭐⭐ | 预估时间：30 秒

### 初始状态

```python
print("hello world")
```

### 目标状态

```python
print("")
```

---

### 操作方法

```vim
di"
```

---

### 关键命令解析

| 命令  | 作用         |
| ----- | ------------ |
| `di"` | 删除双引号内 |
| `di'` | 删除单引号内 |
| `di`  | 删除反引号内 |

---

### 检验清单

- [ ] 知道 `di"` 和 `di'` 的区别
- [ ] 能处理各种引号类型

---

## 删除操作速查

| 操作       | 命令  | 说明         |
| ---------- | ----- | ------------ |
| 删除字符   | `x`   | 光标下字符   |
| 删除前行   | `X`   | 光标前字符   |
| 删除单词   | `diw` | 整个单词     |
| 删除到行尾 | `D`   | 从光标到行尾 |
| 删除到行首 | `d0`  | 从行首到光标 |
| 删除行     | `dd`  | 整行         |
| 删除括号内 | `di(` | 括号内容     |
| 删除引号内 | `di"` | 引号内容     |
