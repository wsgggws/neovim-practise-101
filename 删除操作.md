## 24. 删除单词

```text
hello beautiful world
```

=>

```text
hello world
```

## 25. 删除到行尾

```text
hello world and everyone
```

=> 光标在 "and" 上，删除到行尾

```text
hello world
```

## 26. 删除到行首

```text
hello world and everyone
```

=> 光标在 "and" 上，删除到行首

```text
and everyone
```

## 27. 删除括号内内容

```text
function(arg1, arg2, arg3)
```

=>

```text
function()
```

## 28. 删除引号内内容

```text
print("hello world")
```

=>

```text
print("")
```
