## 21. 复制整行

```text
hello world
neovim is great
```

=> 复制第一行到第二行下方

```text
hello world
neovim is great
hello world
```

## 22. 复制多行

```text
line1
line2
line3
line4
```

=> 复制前两行到末尾

```text
line1
line2
line3
line4
line1
line2
```

## 23. 复制到指定位置

```text
def hello():
    pass

def world():
    pass
```

=> 复制 hello 函数内容到 world 函数中

```text
def hello():
    pass

def world():
    pass
```
