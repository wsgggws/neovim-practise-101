# 跳转导航 (61-63)

> ⭐⭐ 中等 | 分类：代码导航 | 预估时间：3 分钟

### 场景描述

高效导航是阅读和编辑大代码库的基础，本章节覆盖常用跳转技巧。

---

## 61. 跳转到匹配括号

> 难度：⭐ | 预估时间：30 秒

### 初始状态

```python
def hello(name, age):
    if name:
        print(f"Hello {name}")
        return True
    return False
```

---

### 操作方法

```vim
%  " 光标在括号上时使用
```

---

### 关键命令解析

| 命令 | 作用             |
| ---- | ---------------- |
| `%`  | 跳转到匹配的括号 |
| `%`  | 再次按返回       |

---

### 可匹配的符号

| 符号             | 示例       |
| ---------------- | ---------- |
| `()`             | 圆括号     |
| `[]`             | 方括号     |
| `{}`             | 花括号     |
| `#if` / `#endif` | C 预处理器 |

## 62. 跳转到函数定义

> 难度：⭐⭐ | 预估时间：1 分钟

### 初始状态

```python
result = calculate(10, 20)

def calculate(a, b):
    return a + b
```

---

### 操作方法

**方法一：LSP 跳转（推荐）**

```vim
gd  " 跳转到定义
Ctrl-o " 返回上一位置
```

**方法二：Telescope**

```vim
<leader>gd
```

---

### 关键命令解析

| 命令     | 作用         |
| -------- | ------------ |
| `gd`     | 跳转到定义   |
| `Ctrl-o` | 返回上一位置 |
| `Ctrl-i` | 前进位置     |

---

- [ ] 知道如何返回

## 63. 跳转到下一个错误

> 难度：⭐⭐ | 预估时间：1 分钟

### 初始状态

```python
def hello()  # 缺少冒号
    print("hello")
    return True  # 缩进错误
```

---

### 操作方法

**方法一：LSP 跳转**

```vim
]d  " 下一个诊断
[d  " 上一个诊断
```

**方法二：Telescope**

```vim
<leader>fd
```

---

### 关键命令解析

| 命令 | 作用       |
| ---- | ---------- |
| `]d` | 下一个诊断 |
| `[d` | 上一个诊断 |
