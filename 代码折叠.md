# 代码折叠 (64-66)

> ⭐⭐ 中等 | 分类：代码导航 | 预估时间：2 分钟

### 场景描述

代码折叠可以隐藏不重要的代码，专注于当前逻辑。

---

## 64. 折叠函数

> 难度：⭐⭐ | 预估时间：30 秒

### 初始状态

```python
def long_function():
    line1
    line2
    line3
    line4
    line5
    return result
```

### 目标状态

```python
def long_function(): +--- 6 lines ---
```

---

### 操作方法

**方法一：zc（推荐）**

```vim
zc  " 关闭折叠
zo  " 打开折叠
```

**方法二：za**

```vim
za  " 切换折叠
```

---

### 关键命令解析

| 命令 | 作用     |
| ---- | -------- |
| `zc` | 关闭折叠 |
| `zo` | 打开折叠 |
| `za` | 切换折叠 |
| `zM` | 关闭所有 |
| `zR` | 打开所有 |

---

### 检验清单

- [ ] 能折叠和展开代码
- [ ] 知道 `zM` 和 `zR` 的作用

## 65. 折叠代码块

> 难度：⭐⭐ | 预估时间：1 分钟

### 初始状态

```python
class MyClass:
    def method1(self):
        pass

    def method2(self):
        pass

    def method3(self):
        pass
```

---

### 操作方法

**方法一：可视模式**

```vim
Vjjzf
```

**方法二：标记+折叠**

```vim
ma
mb
:'a,'bzf
```

---

### 关键命令解析

| 命令 | 作用     |
| ---- | -------- |
| `zf` | 创建折叠 |
| `zd` | 删除折叠 |

---

### 检验清单

- [ ] 能手动创建折叠
- [ ] 知道如何删除折叠

## 66. 折叠注释块

> 难度：⭐⭐ | 预估时间：30 秒

### 初始状态

```python
"""
This is a long
multi-line
documentation
string
"""
```

---

### 操作方法

```vim
zf
```

---

### 进阶扩展

**折叠级别：**

```vim
:set foldlevel=0  " 只折叠最高级别
:set foldlevel=3  " 折叠到第 3 级
```
