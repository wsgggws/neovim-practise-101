# 分屏操作 (73-75)

> ⭐⭐ 中等 | 提升效率

## 73. 水平分屏比较

**初始状态**：两个独立文件

**操作提示**：

```vim
" 水平分屏打开比较
:diffsplit filename.txt

" 快捷键（LazyVim）
<leader>wd  " 关闭分屏
```

**目标状态**：上下并排显示两个文件，高亮显示差异

---

## 74. 垂直分屏编辑

**初始状态**：

```python
# 文件 1: model.py
class User:
    pass
```

**操作提示**：

```vim
" 垂直分屏
:vsplit filename
" 或
Ctrl-w v

" 切换分屏
Ctrl-w h/j/k/l
```

**目标状态**：左右分屏显示，同时编辑两个文件

---

## 75. 在分屏间移动内容

**初始状态**：

```python
# 左侧文件
def helper():
    pass

# 右侧文件
# 需要 helper 函数
```

**操作提示**：

```vim
" 可视模式选中，移动到另一个分屏
Vjj  " 选中 3 行
Ctrl-w L  " 移动到右侧分屏
```

**目标状态**：helper 函数已移动到右侧文件
