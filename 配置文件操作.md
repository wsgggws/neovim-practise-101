## 87. ENV 文件格式化

```text
DATABASE_URL=localhost
PORT=3000
DEBUG=true
```

=>

```bash
export DATABASE_URL="localhost"
export PORT="3000"
export DEBUG="true"
```

## 88. YAML 缩进修复

```yaml
parent:
child1: value1
child2: value2
```

=>

```yaml
parent:
  child1: value1
  child2: value2
```

## 89. 配置项对齐

```text
name=John
email=john@example.com
age=30
```

=>

```text
name  = John
email = john@example.com
age   = 30
```
