# 配置文件操作 (87-89)

> ⭐⭐ 中等 | 配置管理

## 87. ENV 文件格式化

**初始状态**：

```text
DATABASE_URL=localhost
PORT=3000
DEBUG=true
```

**操作提示**：

```vim
:%s/\([^=]*\)=\(.*\)/export \1="\2"/
```

**目标状态**：

```bash
export DATABASE_URL="localhost"
export PORT="3000"
export DEBUG="true"
```

---

## 88. YAML 缩进修复

**初始状态**：

```yaml
parent:
child1: value1
child2: value2
```

**操作提示**：

```vim
:%s/^  */  /g
" 或使用格式化
```

**目标状态**：

```yaml
parent:
  child1: value1
  child2: value2
```

---

## 89. 配置项对齐

**初始状态**：

```text
name=John
email=john@example.com
age=30
```

**操作提示**：

```vim
:%!column -t -s'='
```

**目标状态**：

```text
name   = John
email  = john@example.com
age    = 30
```
