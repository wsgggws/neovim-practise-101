=>

```python
def calculate_sum(x: int, y: int, z: int) -> int:
    """
    Calculate the sum of positive numbers.

    Args:
        x: First number
        y: Second number
        z: Third number

    Returns:
        Sum of positive numbers
    """
    if x <= 0:
        return 0

    if y <= 0:
        return x

    if z <= 0:
        return x + y

    return x + y + z
```

1. 修复缩进

3. 添加类型提示
