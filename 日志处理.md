# 日志处理 (81-83)

> ⭐⭐ 中等 | 日志分析

## 81. 提取错误日志

**初始状态**：

```text
[INFO] Application started
[ERROR] Connection failed
[INFO] Retry attempt 1
[ERROR] Timeout occurred
[INFO] Application stopped
```

**操作提示**：

```vim
:g/\[ERROR\]/yank A
" 或
:v/\[ERROR\]/d
```

**目标状态**：

```text
[ERROR] Connection failed
[ERROR] Timeout occurred
```

---

## 82. 日志时间戳对齐

**初始状态**：

```text
2024-01-01 10:00:00 Start
2024-01-01 10:05:30 Process
2024-01-01 10:10:15 Complete
```

**操作提示**：

```vim
:%!column -t
" 或
:%s/\d\{4\}-\d\{2\}-\d\{2\}\s*/      /e
```

---

## 83. 过滤特定日志级别

**初始状态**：

```text
DEBUG: detail info
INFO: normal info
WARN: warning message
ERROR: error occurred
```

**操作提示**：

```vim
:v/^\(WARN\|ERROR\)/d
```

**目标状态**：

```text
WARN: warning message
ERROR: error occurred
```

---
