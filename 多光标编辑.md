## 29. 批量修改相同单词

```python
name = "John"
print(name)
return name
```

=>

```python
user = "John"
print(user)
return user
```

## 30. 列编辑模式

```text
apple
banana
cherry
date
```

=>

```text
- apple
- banana
- cherry
- date
```

## 31. 多行行尾添加

```text
const a = 1
const b = 2
const c = 3
```

=>

```text
const a = 1;
const b = 2;
const c = 3;
```
