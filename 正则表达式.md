## 98. 匹配并替换模式

```text
file_2024_01_01.txt
file_2024_02_15.txt
file_2024_03_30.txt
```

=>

```text
file_2024-01-01.txt
file_2024-02-15.txt
file_2024-03-30.txt
```

## 99. 提取匹配组

```text
User: John (age: 30)
User: Jane (age: 25)
User: Bob (age: 35)
```

=>

```text
John is 30 years old
Jane is 25 years old
Bob is 35 years old
```

## 100. 条件替换

```text
TODO: implement feature A
DONE: implement feature B
TODO: implement feature C
```

=>

```text
[ ] implement feature A
[x] implement feature B
[ ] implement feature C
```
