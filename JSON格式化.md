## 35. JSON 压缩格式化

```json
{
  "name": "John",
  "age": 30,
  "city": "New York"
}
```

=>

```json
{ "name": "John", "age": 30, "city": "New York" }
```

## 36. JSON 美化格式化

```json
{ "name": "John", "age": 30, "city": "New York" }
```

=>

```json
{
  "name": "John",
  "age": 30,
  "city": "New York"
}
```

## 37. 修复 JSON 逗号

```json
{
  "name": "John"
  "age": 30
  "city": "New York"
}
```

=>

```json
{
  "name": "John",
  "age": 30,
  "city": "New York"
}
```
