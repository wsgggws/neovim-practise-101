# 第 07 章：工作流与环境管理

> 本章涵盖 102-123 题，学习 Buffer、标签页、文件管理等。

## 核心技能

| 技能        | 命令                | 说明        |
| ----------- | ------------------- | ----------- |
| Buffer 切换 | `:bnext` / `:bprev` | 切换 Buffer |
| Buffer 关闭 | `:bd`               | 删除 Buffer |
| 标签页管理  | `:tabnew`           | 新建标签页  |
| 会话保存    | `:mksession`        | 保存会话    |

## 关键命令速查

```vim
" Buffer 操作
:ls           " 列出 Buffer
:bnext        " 下个 Buffer
:bprev        " 上个 Buffer
:bfirst       " 第一个
:blast        " 最后一个
:bd           " 关闭当前
%bd           " 关闭所有

" 标签页操作
:tabnew       " 新建标签页
:tabclose     " 关闭标签页
:tabnext      " 下个标签页
:tabprevious  " 上个标签页
gt            " 切换标签页
gT            " 上一个标签页

" 文件操作
:e file       " 打开文件
:w            " 保存
:saveas file  " 另存为
:find file    " 搜索打开

" 目录管理
:!mkdir -p    " 创建目录
:Rex           " 文件浏览器
:Explore      " 打开目录
```

## 常见场景

1. **多文件编辑**：Buffer 切换
2. **项目切换**：标签页管理
3. **会话恢复**：mksession
4. **文件搜索**：Telescope find_files

## 插件推荐

- **Telescope**：文件搜索、Buffer 切换
- **Harpoon**：快速文件切换
- **bufexplorer**：Buffer 浏览器
- **vim-startify**：启动画面
