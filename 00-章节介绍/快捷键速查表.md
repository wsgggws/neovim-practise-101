# Neovim 快捷键速查表

> 练习前快速查阅

## 模式切换

| 按键     | 功能                       |
| -------- | -------------------------- |
| `Esc`    | 返回 Normal 模式           |
| `i`      | 进入 Insert 模式（光标前） |
| `a`      | 进入 Insert 模式（光标后） |
| `o`      | 在下方插入新行             |
| `O`      | 在上方插入新行             |
| `:`      | 进入命令行模式             |
| `v`      | 进入可视模式（字符）       |
| `V`      | 进入可视模式（行）         |
| `Ctrl-v` | 进入可视模式（列）         |

## 移动光标

### 基础移动

| 按键 | 功能           |
| ---- | -------------- |
| `h`  | 左             |
| `j`  | 下             |
| `k`  | 上             |
| `l`  | 右             |
| `w`  | 下一个单词开头 |
| `b`  | 上一个单词开头 |
| `e`  | 下一个单词结尾 |
| `ge` | 上一个单词结尾 |

### 行内移动

| 按键 | 功能         |
| ---- | ------------ |
| `0`  | 行首         |
| `$`  | 行尾         |
| `^`  | 行首非空字符 |
| `g_` | 行尾非空字符 |

### 文件内移动

| 按键 | 功能          |
| ---- | ------------- |
| `gg` | 文件开头      |
| `G`  | 文件结尾      |
| `:n` | 跳转到第 n 行 |
| `nG` | 跳转到第 n 行 |

### 屏幕移动

| 按键     | 功能       |
| -------- | ---------- |
| `Ctrl-e` | 向下滚屏   |
| `Ctrl-y` | 向上滚屏   |
| `Ctrl-d` | 向下半屏   |
| `Ctrl-u` | 向上半屏   |
| `zt`     | 当前行置顶 |
| `zz`     | 当前行居中 |
| `zb`     | 当前行置底 |

## 编辑操作

### 插入模式快捷键

| 按键     | 功能           |
| -------- | -------------- |
| `Ctrl-h` | 删除前一个字符 |
| `Ctrl-w` | 删除前一个单词 |
| `Ctrl-u` | 删除当前行     |
| `Ctrl-t` | 缩进           |
| `Ctrl-d` | 取消缩进       |

### 删除

| 按键  | 功能           |
| ----- | -------------- |
| `x`   | 删除光标下字符 |
| `X`   | 删除光标前字符 |
| `d`   | 配合移动命令   |
| `dd`  | 删除整行       |
| `d$`  | 删除到行尾     |
| `d0`  | 删除到行首     |
| `diw` | 删除单词       |
| `dap` | 删除段落       |

### 修改

| 按键  | 功能         |
| ----- | ------------ |
| `r`   | 替换单个字符 |
| `R`   | 进入替换模式 |
| `c`   | 配合移动命令 |
| `cc`  | 修改整行     |
| `ciw` | 修改单词     |
| `ci"` | 修改引号内   |
| `ci(` | 修改括号内   |

### 复制粘贴

| 按键  | 功能         |
| ----- | ------------ |
| `y`   | 复制（yank） |
| `yy`  | 复制整行     |
| `yiw` | 复制单词     |
| `p`   | 粘贴到光标后 |
| `P`   | 粘贴到光标前 |
| `xp`  | 交换两个字符 |

### 撤销重做

| 按键     | 功能     |
| -------- | -------- |
| `u`      | 撤销     |
| `Ctrl-r` | 重做     |
| `U`      | 撤销整行 |

## 可视模式

| 按键     | 功能         |
| -------- | ------------ |
| `v`      | 进入字符可视 |
| `V`      | 进入行可视   |
| `Ctrl-v` | 进入列可视   |
| `o`      | 切换选区端点 |
| `gv`     | 重新选择     |
| `y`      | 复制选中     |
| `d`      | 删除选中     |
| `c`      | 修改选中     |

## 搜索与替换

### 搜索

| 按键       | 功能               |
| ---------- | ------------------ |
| `/pattern` | 向下搜索           |
| `?pattern` | 向上搜索           |
| `n`        | 下一个匹配         |
| `N`        | 上一个匹配         |
| `*`        | 搜索光标下单词     |
| `#`        | 反向搜索光标下单词 |

### 替换

| 命令               | 功能           |
| ------------------ | -------------- |
| `:s/old/new`       | 替换当前行     |
| `:s/old/new/g`     | 替换当前行全部 |
| `:%s/old/new/g`    | 替换文件全部   |
| `:n,m s/old/new/g` | 替换 n-m 行    |

## 窗口操作

| 按键             | 功能                  |
| ---------------- | --------------------- | ---------- |
| `Ctrl-w s`       | 水平分割              |
| `Ctrl-w v`       | 垂直分割              |
| `Ctrl-w q`       | 关闭窗口              |
| `Ctrl-w w`       | 切换窗口              |
| `Ctrl-w h/j/k/l` | 切换到左/下/上/右窗口 |
| `Ctrl-w =`       | 等分窗口              |
| `Ctrl-w _`       | 最大化高度            |
| `Ctrl-w          |`                     | 最大化宽度 |

## Buffer 和标签页

| 按键      | 功能            |
| --------- | --------------- |
| `:bn`     | 下一个 Buffer   |
| `:bp`     | 上一个 Buffer   |
| `:bd`     | 关闭当前 Buffer |
| `:tabnew` | 新建标签页      |
| `gt`      | 下一个标签页    |
| `gT`      | 上一个标签页    |

## 宏录制

| 按键 | 功能         |
| ---- | ------------ |
| `qa` | 开始录制到 a |
| `q`  | 停止录制     |
| `@a` | 执行宏 a     |
| `@@` | 重复上次宏   |

## 常用组合技

| 操作       | 快捷键 | 说明           |
| ---------- | ------ | -------------- |
| 复制整行   | `yyp`  | yy 复制 p 粘贴 |
| 删除整行   | `ddp`  | dd 删除 p 交换 |
| 大小写转换 | `gUU`  | 整行转大写     |
| 大小写转换 | `guu`  | 整行转小写     |
| 注释代码   | `gcc`  | 注释/取消注释  |
| 格式化缩进 | `==`   | 当前行缩进     |
| 快速退出   | `ZZ`   | 保存并退出     |
| 放弃修改   | `ZQ`   | 不保存退出     |
