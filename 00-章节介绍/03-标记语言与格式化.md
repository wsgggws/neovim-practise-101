# 第 03 章：标记语言与格式化

> 本章涵盖 41-60 题，学习 HTML、Markdown、数字和格式转换。

## 核心技能

| 技能          | 命令           | 说明         |
| ------------- | -------------- | ------------ |
| HTML 标签修改 | `cit`          | 修改标签内容 |
| Markdown 链接 | `viwS[`        | 包裹为链接   |
| 数字递增      | `Ctrl-a`       | 数字+1       |
| 格式转换      | `:%s/.../.../` | 正则替换     |

## 关键命令速查

````vim
" HTML 操作
cit    " change inside tag
dat    " delete a tag
ysiw<div> " 包裹为 div

" Markdown
viwS[  " 创建链接
V3j:s/^/- / " 批量列表
viwS``` " 创建代码块

" 数字操作
Ctrl-a  " 递增
Ctrl-x  " 递减
:let i=1 | %s/\d\+/\=i/ | let i+=1 " 批量递增

" 格式转换
:%s/_\([a-z]\)/\U\1/g  " 蛇形转驼峰
:%s/\([A-Z]\)/_\l\1/g  " 驼峰转蛇形
````

## 常见场景

1. **批量修改 HTML**：使用 `cit` 和 `ysiw`
2. **文档格式整理**：Markdown 列表、代码块
3. **代码风格转换**：驼峰/蛇形/常量
4. **数据处理**：CSV/JSON 转换
