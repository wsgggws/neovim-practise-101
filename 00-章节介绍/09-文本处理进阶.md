# 第 09 章：文本处理进阶

> 本章涵盖 139-151 题，学习高级文本对齐、列编辑、表达式计算等。

## 核心技能

| 技能       | 命令                           | 说明       |
| ---------- | ------------------------------ | ---------- |
| 列编辑     | `Ctrl-v`                       | 可视块模式 |
| 对齐       | `:Left` / `:Center` / `:Right` | 文本对齐   |
| 表达式计算 | `Ctrl-r =`                     | 计算表达式 |
| 序列生成   | `Ctrl-a`                       | 数字递增   |

## 关键命令速查

```vim
" 列编辑
Ctrl-v       " 进入可视块
I...<Esc>    " 批量插入
A...<Esc>    " 批量追加
c            " 批量修改
d            " 批量删除

" 文本对齐
:Left        " 左对齐
:Center      " 居中
:Right       " 右对齐
:Tabularize /|  " 按 | 对齐

" 表达式计算
Ctrl-r =     " 插入计算结果
5+5<Enter>   " 计算 5+5
line('.')    " 当前行号

" 序列生成
Ctrl-a       " 递增
Ctrl-x       " 递减
V选中 =      " 批量计算
```

## 常见场景

1. **表格对齐**：列编辑 + 对齐命令
2. **数据转换**：CSV 处理
3. **代码格式化**：对齐等号
4. **序列生成**：编号、行号

## 高级技巧

```vim
" 批量替换公式
:%s/\d\+/\=submatch(0)+1/g  " 数字+1

" 生成序列
:let i=1 | g/^/s//\=i/ | let i+=1

" 复杂对齐
:Tabularize /,\zs/l1
```
