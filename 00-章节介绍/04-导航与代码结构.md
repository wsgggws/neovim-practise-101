# 第 04 章：导航与代码结构

> 本章涵盖 61-80 题，学习跳转、折叠、Git 操作等。

## 核心技能

| 技能      | 命令                 | 说明          |
| --------- | -------------------- | ------------- |
| 跳转函数  | `gd`                 | 跳转到定义    |
| 跳回      | `Ctrl-o`             | 返回上一位置  |
| 代码折叠  | `za`                 | 打开/关闭折叠 |
| Git blame | `:Gitsigns blame`    | 查看修改信息  |
| Git diff  | `:Gitsigns diffthis` | 查看差异      |

## 关键命令速查

```vim
" 跳转导航
gd     " 跳转到定义
gf     " 跳转到文件
Ctrl-o " 返回
Ctrl-i " 前进
]d     " 下一个诊断
[d     " 上一个诊断

" 代码折叠
za     " 打开/关闭当前
zM     " 全部关闭
zR     " 全部打开
zf     " 创建折叠

" Git 操作
:Gitsigns blame      " blame 当前行
:Gitsigns diffthis   " 显示差异
<leader>gc           " Telescope 冲突列表

" 分屏
:sp file             " 水平分屏
:vsp file            " 垂直分屏
Ctrl-w h/j/k/l       " 切换窗口
```

## 常见场景

1. **阅读大型代码库**：善用折叠和跳转
2. **代码审查**：使用 Git diff
3. **解决冲突**：定位并处理合并冲突
4. **多文件对比**：分屏操作
