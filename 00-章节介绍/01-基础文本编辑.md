# 章节 1：基础文本编辑

> 本章练习：1-6

## 核心命令速查

| 操作       | 命令                                       | 说明                 |
| ---------- | ------------------------------------------ | -------------------- |
| 删除空行   | `:g/^$/d`                                  | 全局删除空行         |
| 去重排序   | `:sort u`                                  | 排序并去重           |
| 行排序     | `:sort [n]`                                | 可选数字排序         |
| 倒序       | `:g/^/m 0`                                 | 每行移到开头         |
| 递增序号   | `:let i=x \| g/^/s//\=i." "/ \| let i=i+1` | 生成序号             |
| 修改引号内 | `ci"`                                      | Change inside quotes |

## 重点概念

### 1. 全局命令 `:g`

- `:g/pattern/command` - 对所有匹配行执行命令
- `^$` - 匹配空行（行首和行尾之间无内容）

### 2. 排序命令 `:sort`

- `:sort` - 文本排序
- `:sort n` - 数字排序
- `:sort u` - 排序并去重
- `:sort!` - 降序

### 3. 移动命令 `:m`

- `:m 0` - 移到文件开头
- `:m $` - 移到文件末尾
- `:g/^/m 0` - 倒序所有行

### 4. 文本对象 `ci"`, `ci'`, `ci(`

- `c` - change 修改
- `i` - inside 内部
- `"` / `'` / `(` - 环绕符号

## 练习路线

1. **删除空行** → 学会使用 `:g/^$/d`
2. **删除重复行** → 掌握 `:sort u`
3. **添加序号** → 理解 `:g` 配合变量
4. **倒置行序** → 学会 `:g/^/m 0`
5. **行排序** → 掌握 `:sort` 各种选项
6. **修改文本对象** → 基础编辑器的核心技能

## 常见问题

**Q: 为什么我的空行没删掉？**
A: 检查是否包含空格 `:g/^\s*$/d`

**Q: sort 命令会改变原始顺序吗？**
A: 是的，sort 会重新排序行

**Q: 如何只对部分行排序？**
A: 先可视选中 `:'<,'>sort`
