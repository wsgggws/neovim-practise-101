# Neovim 实战 101 系列介绍

> 181 道练习题，从入门到进阶

## 课程结构

| 阶段 | 章节              | 题目范围 | 难度   |
| ---- | ----------------- | -------- | ------ |
| 入门 | 基础文本编辑      | 1-20     | ⭐     |
| 进阶 | 编辑操作技巧      | 21-60    | ⭐⭐   |
| 高级 | 导航与代码结构    | 61-100   | ⭐⭐⭐ |
| 专家 | 终极挑战 & 工作流 | 101-181  | ⭐⭐⭐ |

## 学习建议

### 新手入门（1-20）

1. **每天 5 题**：不要贪多，循序渐进
2. **动手实践**：复制代码到 Neovim 实操
3. **理解原理**：知道命令为什么这样工作
4. **记录笔记**：整理自己的快捷键手册

### 进阶提升（21-60）

- 掌握多光标、宏录制、文本对象
- 理解 LSP 和代码重构
- 熟悉各种格式化操作

### 高手修炼（61-181）

- Buffer 和标签页管理
- 分屏操作与窗口导航
- 命令行高级应用
- 完整工作流整合

## 练习方法

```
每道题目的正确练习流程：

1. 复制「初始状态」到 Neovim
2. 思考：这道题有几种解法？
3. 尝试用最少的按键完成
4. 查看「操作提示」对比解法
5. 记录：不熟悉的命令加入个人笔记
6. 复习：隔天重新练习巩固
```

## 核心快捷键速查

### 移动

| 按键      | 功能              |
| --------- | ----------------- |
| `h/j/k/l` | 左/下/上/右       |
| `w/b`     | 下一个/上一个单词 |
| `0/$`     | 行首/行尾         |
| `gg/G`    | 文件开头/结尾     |

### 编辑

| 按键  | 功能              |
| ----- | ----------------- |
| `i/a` | 进入插入模式      |
| `x/d` | 删除字符/更多     |
| `y/p` | 复制/粘贴         |
| `c`   | 修改（删除+插入） |

### 模式

| 按键         | 功能           |
| ------------ | -------------- |
| `Esc`        | 返回Normal模式 |
| `v/V/Ctrl-v` | 可视/行选/列选 |
| `:`          | 命令行模式     |

## 常见问题

**Q: 需要背快捷键吗？**
A: 不需要死记硬背。用的多了自然记住，重点是理解 Vim 的操作逻辑。

**Q: 练习要坚持多久？**
A: 建议至少 2 周，每天 30 分钟。建立肌肉记忆需要时间。

**Q: 跟不上进度怎么办？**
A: 放慢速度，确保每道题都理解透彻。质量比数量重要。

**Q: 视频在哪里看？**
A: B站/YouTube 搜索 [频道名]
