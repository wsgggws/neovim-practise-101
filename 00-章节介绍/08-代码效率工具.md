# 第 08 章：代码效率工具

> 本章涵盖 124-138 题，学习 Snippets（代码片段）的使用和创建。

## 核心技能

| 技能       | 命令        | 说明           |
| ---------- | ----------- | -------------- |
| 展开片段   | `Tab`       | 触发代码片段   |
| 选择占位符 | `Ctrl-n`    | 下一个占位符   |
| 跳转占位符 | `Ctrl-j`    | 跳到下一占位符 |
| 创建片段   | `:Snippets` | 编辑代码片段   |

## 关键命令速查

```vim
" Luasnip（LazyVim 默认）
Tab             " 展开/跳转
Shift-Tab       " 上一个占位符
Ctrl-j          " 强制跳转
Ctrl-k          " 显示参数

" 常用片段
for             " for 循环
while           " while 循环
if              " if 语句
fun             " 函数定义
cl              " console.log
```

## 常见场景

1. **快速输入循环**：for、while 片段
2. **函数生成**：fun、function 片段
3. **日志输出**：cl、console.log 片段
4. **React 组件**：组件模板片段

## 片段语法示例

```snippets
# 代码片段定义
snippet if "if statement" b
if ${1:condition} then
 ${2:body}
end
endsnippet

snippet fun "function" b
function ${1:name}(${2:args})
 ${3:body}
end
endsnippet
```

## 插件推荐

- **LuaSnip**：代码片段引擎
- **friendly-snippets**：常用片段集合
- **vim-vsnip**：VS Code 风格片段
