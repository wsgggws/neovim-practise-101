# 第 02 章：编辑操作技巧

> 本章涵盖 21-40 题，重点学习复制、删除、多光标等编辑技巧。

## 核心技能

| 技能     | 命令      | 说明           |
| -------- | --------- | -------------- |
| 复制整行 | `yy`      | 复制当前行     |
| 复制多行 | `Nyy`     | 复制 N 行      |
| 粘贴     | `p` / `P` | 粘贴到下/上方  |
| 删除单词 | `dw`      | 删除到词尾     |
| 更改单词 | `cw`      | 更改到词尾     |
| 多光标   | `Ctrl-n`  | 选中下一个匹配 |

## 关键命令速查

```vim
" 复制粘贴
yy    " 复制整行
p     " 粘贴到光标后
P     " 粘贴到光标前

" 删除
dw    " 删除单词
dd    " 删除整行
d$    " 删除到行尾
d^    " 删除到行首

" 多光标
Ctrl-n " 选中下一个
Ctrl-p " 取消当前选中
n     " 跳到下一个
N     " 跳到上一个
```

## 常见场景

1. **批量修改变量名**：多光标 + 重命名
2. **删除多余空行**：`:g/^$/d`
3. **代码格式化**：宏录制 + 重复执行

- 尝试不同的组合方式
- 在实际代码中使用这些技巧
