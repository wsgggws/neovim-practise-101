## 78. 数组转多行

```javascript
const arr = [1, 2, 3, 4, 5];
```

=>

```javascript
const arr = [1, 2, 3, 4, 5];
```

## 79. 多行转数组

```javascript
const arr = [1, 2, 3];
```

=>

```javascript
const arr = [1, 2, 3];
```

## 80. 对象属性排序

```javascript
const obj = {
  z: 1,
  a: 2,
  m: 3,
};
```

=>

```javascript
const obj = {
  a: 2,
  m: 3,
  z: 1,
};
```
