# 包裹文本 (14-16)

> ⭐ 简单 | 分类：基础编辑 | 预估时间：2 分钟

### 场景描述

使用 vim-surround 插件可以快速包裹文本，适合代码开发和格式化。

---

## 14. 用引号包裹单词

> 难度：⭐ | 预估时间：30 秒

### 初始状态

```text
hello world
```

### 目标状态

```text
"hello" "world"
```

---

### 操作方法

**方法一：viwS"（推荐）**

```vim
viwS"
```

**方法二：yiwo"**

```vim
yiwo"
```

---

### 关键命令解析

| 命令  | 作用         |
| ----- | ------------ |
| `viw` | 可视选中单词 |
| `S"`  | 用 " 包裹    |

## 15. 用括号包裹

## 16. 更改包裹符号

> 难度：⭐⭐ | 预估时间：1 分钟

### 初始状态

```text
"hello world"
```

### 目标状态

```text
'hello world'
```

---

### 操作方法

**方法一：cs"'（推荐）**

```vim
cs"'
```

**方法二：ds" 然后 S'**

```vim
ds"
S'
```

---

### 关键命令解析

| 命令 | 作用               |
| ---- | ------------------ |
| `cs` | change surrounding |
| `ds` | delete surrounding |

---

### 解法对比

| 方法     | 命令         | 按键次数 |
| -------- | ------------ | -------- |
| 直接修改 | `cs"'`       | 3        |
| 两步     | `ds"` + `S'` | 4        |

---

### 检验清单

- [ ] 知道 `cs` 的用法
- [ ] 能快速更换包裹符号

## vim-surround 速查

| 命令    | 功能               |
| ------- | ------------------ |
| `ysiw"` | 包裹单词为双引号   |
| `yss(`  | 包裹整行为括号     |
| `cs"'`  | 修改双引号为单引号 |
| `cs"(`  | 修改为括号         |
| `ds"`   | 删除双引号         |
| `S"`    | 可视模式选中后包裹 |

### 可用包裹符号

| 符号    | 示例      |
| ------- | --------- |
| `'`     | 单引号    |
| `"`     | 双引号    |
| `(` `)` | 圆括号    |
| `[` `]` | 方括号    |
| `{` `}` | 花括号    |
| `<` `>` | 尖括号    |
| `` ` `` | 反引号    |
| `t`     | HTML 标签 |
