## 17. 简单查找替换

```text
hello world
hello neovim
hello vim
```

=>

```text
hi world
hi neovim
hi vim
```

## 18. 替换当前行

```text
hello world hello neovim hello vim
```

=>

```text
hi world hi neovim hi vim
```

## 19. 大小写敏感替换

```text
Hello world
hello neovim
HELLO vim
```

=>

```text
Hi world
hi neovim
HELLO vim
```

## 20. 正则表达式替换

```text
file1.txt
file2.txt
file3.txt
```

=>

```text
file1.md
file2.md
file3.md
```
