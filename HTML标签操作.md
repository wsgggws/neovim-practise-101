## 38. 修改 HTML 标签

```html
<div>Hello World</div>
```

=>

```html
<span>Hello World</span>
```

## 39. 删除 HTML 标签

```html
<div>Hello World</div>
```

=>

```html
Hello World
```

## 40. 包裹 HTML 标签


=>

```html
<div>Hello World</div>
```

## 41. 修改标签属性

```html
<div class="old">Content</div>
```

=>

```html
<div class="new">Content</div>
```
