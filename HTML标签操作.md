# HTML 标签操作 (38-41)

> ⭐⭐ 中等 | 分类：Web 开发 | 预估时间：2 分钟

### 场景描述

使用 vim-surround 快速操作 HTML 标签。

---

## 38. 修改 HTML 标签

> 难度：⭐⭐ | 预估时间：30 秒

### 初始状态

```html
<div>Hello World</div>
```

### 目标状态

```html
<span>Hello World</span>
```

---

### 操作方法

```vim
cit<div>
```

---

### 关键命令解析

| 命令  | 作用              |
| ----- | ----------------- |
| `cit` | change inside tag |

---

### 检验清单

- [ ] 知道 `cit` 的用法
- [ ] 能快速修改标签名

---

## 39. 删除 HTML 标签

> 难度：⭐⭐ | 预估时间：30 秒

### 初始状态

```html
<div>Hello World</div>
```

### 目标状态

```html
Hello World
```

---

### 操作方法

```vim
dat  " delete a tag
dit  " delete inside tag
```

---

### 解法对比

| 命令  | 作用           |
| ----- | -------------- |
| `dat` | 删除标签及内容 |
| `dit` | 只删除内容     |

---

### 检验清单

- [ ] 知道 `dat` 和 `dit` 的区别
- [ ] 能根据需求选择

---

## 40. 包裹 HTML 标签

> 难度：⭐⭐ | 预估时间：30 秒

### 初始状态

```html
Hello World
```

### 目标状态

```html
<div>Hello World</div>
```

---

### 操作方法

```vim
ysiw<div>
```

---

### 关键命令解析

| 命令   | 作用                         |
| ------ | ---------------------------- |
| `ysiw` | yank and surround inner word |

---

### 检验清单

- [ ] 能用 `ys` 包裹文本
- [ ] 知道不同标签的用法

---

## 41. 修改标签属性

> 难度：⭐⭐ | 预估时间：30 秒

### 初始状态

```html
<div class="old">Content</div>
```

### 目标状态

```html
<div class="new">Content</div>
```

---

### 操作方法

```vim
cf"new<Esc>
```

---

### 检验清单

- [ ] 能修改标签属性
- [ ] 知道 `cf` 的用法

---

## HTML 操作速查

| 命令        | 功能         |
| ----------- | ------------ |
| `cit`       | 修改标签内   |
| `cat`       | 修改整个标签 |
| `dat`       | 删除标签     |
| `dit`       | 删除标签内   |
| `ysiw<div>` | 包裹为 div   |
