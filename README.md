## 💡 使用建议

1. **循序渐进**：按照分类从基础开始练习
2. **动手实践**：每道题都亲自操作，不要只看不练
3. **重复练习**：对于不熟练的操作多练几次
4. **场景应用**：在实际工作中主动使用学到的技巧
5. **自定义快捷键**：根据个人习惯优化常用操作的快捷键

## 📝 练习方法

每道题目包含：

- **初始文本**：练习的起始状态
- **目标文本**：期望达到的结果
- **操作方法**：关键 Neovim 命令
- **检验清单**：检查是否掌握

建议练习流程：

1. 复制初始文本到 Neovim
2. 尝试用最少的操作达到目标状态
3. 查看答案文件对比方法
4. 理解并记忆关键命令
5. 再次练习直到熟练
