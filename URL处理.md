## 84. 提取 URL 参数

```text
https://example.com?name=John&age=30&city=NYC
```

=>

```text
name: John
age: 30
city: NYC
```

## 85. URL 编码

```text
hello world & special/chars
```

=>

```text
hello%20world%20%26%20special%2Fchars
```

## 86. URL 解码

```text
hello%20world%20%26%20special%2Fchars
```

=>

```text
hello world & special/chars
```
