## 67. 自动导入缺失模块

```python
# 使用了未导入的 os 模块
path = os.path.join("a", "b")
```

=>

```python
import os

path = os.path.join("a", "b")
```

## 68. 修复拼写错误

```python
def caculate(a, b):  # caculate 拼写错误
    return a + b
```

=>

```python
def calculate(a, b):
    return a + b
```

## 69. 添加缺失的类型提示

```python
def add(a, b):
    return a + b
```

=>

```python
def add(a: int, b: int) -> int:
    return a + b
```
